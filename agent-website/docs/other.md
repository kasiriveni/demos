## Objective

Create a Django application with HTML/CSS frontend that serves as a quick prototype (MVP) focused on core functionality. The site should display GitHub repositories alongside Claude AI capability comparisons in an intuitive, visually modern interface.

## Core Features

The application must include:
- A clean header with navigation
- A section comparing GitHub repositories with Claude AI capabilities
- Repository cards displaying key details
- Claude comparison metrics or features
- A footer with links

## Technical Requirements

**Stack:** Python/Django with responsive HTML/CSS frontend

**Scope:** Build for MVP—prioritize core functionality over comprehensive feature coverage. Focus on the essential views and data display needed to validate the concept.

**Design:** Modern web design principles with a contemporary color scheme, good typography, spacing, and visual hierarchy. The interface should feel professional and intuitive.

**Responsiveness:** Ensure the design works well on both desktop and mobile devices. Include responsive considerations for different screen sizes.

**Code Quality:** Produce clean, well-structured code that is easy to customize and maintain. While production-ready in approach, prioritize clarity and extensibility appropriate for an MVP.

## Deliverables

Provide the complete Django application structure (models, views, templates, static CSS) with a fully functional, responsive frontend that can be run immediately. Include any necessary static files and basic styling to achieve the modern aesthetic described.
