# AI Project — Examples & Demos

A workspace of small demos, utilities, and example projects demonstrating core Python, a full-stack TODO app (Express + React), an agent website, and various CLI tools.

Overview of top-level folders:

- `backend`: Express.js TODO API with tests and Docker files.
- `frontend`: React + Vite frontend for the TODO app (Tailwind CSS).
- `agent-website`: Vanilla JS site used for agent demos and Vitest tests.
- `claude-cli-website`: CLI examples and helper scripts.
- `python-examples`: Many small, core-Python example scripts (no frameworks).
- `docs`: Documentation and guides.
- `example-website`: Tiny static example site.
- `project`: Additional project scaffold and examples.

Quick start

1. Browse the examples in `python-examples/` or open the folders for backend/frontend/agent-website.
2. Run a Python example:

```bash
python python-examples/101.py
```

3. Backend (Node) quick run:

```bash
cd backend
npm install
npm run dev
```

See `python-examples/README.md` for an index of the Python examples added to this workspace.
