🟢 Beginner Level

1. Installation & Setup

Get started with Claude CLI installation and basic configuration.

# Install Claude CLI
npm install -g @anthropic-ai/claude-cli

# Verify installation
claude --version

# Set your API key
export ANTHROPIC_API_KEY=your_api_key_here

2. Your First Command

Send your first message to Claude CLI and get a response.

# Simple query
claude "What is Claude CLI?"

# Multi-line prompt
claude "Explain the concept of:
1. APIs
2. REST
3. GraphQL"

3. Basic Flags & Options

Learn essential command-line flags.

# Show help
claude --help

# Set model
claude --model claude-3-opus-20250219 "Your prompt"

# Set temperature (0-1)
claude --temperature 0.5 "Your prompt"

# Set max tokens
claude --max-tokens 1000 "Your prompt"

4. Reading from Files

Process file content with Claude CLI.

# Read single file
claude "Summarize this:" < document.txt

# Process with cat
cat code.js | claude "Review this code"

# Pass file as context
claude "Fix errors in:" "$(cat script.py)"

5. Output Options

Control how results are displayed and saved.

# Save to file
claude "Generate code" > output.txt

# Append to file
claude "More code" >> output.txt

# Format as JSON
claude --json "Your prompt"

# Show tokens used
claude --show-tokens "Your prompt"

🟡 Intermediate Level

1. Conversation Context

Maintain conversation history and context across multiple calls.

# Start a conversation
claude --conversation my-chat "First message"

# Continue conversation
claude --conversation my-chat "Follow-up question"

# List conversations
claude --list-conversations

# View conversation history
claude --show-history my-chat

2. System Prompts

Use system prompts to define Claude's behavior and role.

# Set system prompt
claude --system "You are an expert Python developer" \
"Write a function to sort arrays"

# From file
claude --system-file system-prompt.txt \
"Your user prompt"

# Multi-role system
claude --system "Act as a teacher" \
"Explain quantum computing"

3. Advanced Model Options

Tune model parameters for specific use cases.

# Available models
claude --list-models

# Use specific version
claude --model claude-3-5-sonnet-20241022 "prompt"

# Adjust temperature
# 0 = deterministic, 1 = creative
claude --temperature 0.2 "Code review"
claude --temperature 0.9 "Story writing"

# Max tokens control
claude --max-tokens 2000 "Long response prompt"

4. Batch Processing

Process multiple prompts efficiently.

# From file with multiple queries
cat queries.txt | while read line; do
claude "$line" >> results.txt
done

# Batch with parallel processing
cat items.txt | parallel claude "Process: {}"

# CSV processing
while IFS=, read id query; do
claude "$query" >> output_$id.txt
done < data.csv

5. Error Handling & Validation

Handle errors and validate responses gracefully.

# Check exit code
claude "Your prompt"
if [ $? -ne 0 ]; then
echo "Error occurred"
fi

# Validate API key
claude --validate-key

# Retry on failure
claude --retry 3 "Your prompt"

# Timeout handling
timeout 30s claude "Long task"

🔴 Advanced Level

1. Custom Workflows & Scripting

Build sophisticated automation with Claude CLI.

#!/bin/bash
# Advanced workflow script

analyze_code() {
local file=$1
local model=${2:-claude-3-5-sonnet-20241022}

echo "Analyzing $file..."
claude --model "$model" \
--system "You are a code reviewer" \
"Review this code: $(cat $file)"
}

# Usage
analyze_code "app.js" "claude-3-opus-20250219"

2. Integration with External Tools

Combine Claude CLI with other CLI tools for powerful workflows.

# Process git diffs
git diff | claude --system "Review code changes"

# Analyze logs
tail -f app.log | claude "Diagnose issues"

# Database query analysis
psql -c "EXPLAIN SELECT ..." | \
claude "Optimize this query"

# JSON manipulation
jq '.items[]' data.json | \
claude --json "Structure this data"

3. Multi-Turn Conversations

Build complex interactive conversations with state management.

#!/bin/bash
SESSION="debug-session"

# Start debugging session
claude --conversation "$SESSION" \
--system "You are a debugging expert" \
"I'm seeing 500 errors"

# Continue with context
claude --conversation "$SESSION" \
"The error happens at line 42"

# Save and export
claude --export-conversation "$SESSION" > debug.md

4. Performance Optimization

Optimize for speed, cost, and efficiency.

# Use faster model for simple tasks
claude --model claude-3-5-haiku-20241022 \
--temperature 0 "Quick task"

# Cache context for repeated queries
CONTEXT=$(cat large_file.txt)
for query in query1 query2 query3; do
claude --cache "$CONTEXT" "$query"
done

# Monitor token usage
claude --show-tokens --model claude-3-opus-20250219 \
"Long prompt" | grep -i token

5. Building CLI Tools & Utilities

Create reusable Claude-powered command-line tools.

#!/bin/bash
# claude-translator.sh

usage() {
echo "Usage: $0 --from LANG --to LANG --file FILE"
}

translate() {
local from=$1 to=$2 file=$3
claude --system "Translate from $from to $to" \
"Translate this: $(cat $file)"
}

while [[ $# -gt 0 ]]; do
case $1 in
--from) from=$2; shift 2 ;;
--to) to=$2; shift 2 ;;
--file) file=$2; shift 2 ;;
*) usage; exit 1 ;;
esac
done

translate "$from" "$to" "$file"
